# Dottie MVP 와이어프레임 & 화면 플로우

---

## 화면 플로우 요약

```
[스플래시] → [온보딩 3페이지] → [로그인 (카카오/Apple/Google)]
                                      ↓
                              [닉네임 입력 + 캐릭터 선택]
                                      ↓
                              [위치 권한 요청]
                                      ↓
                    ┌─────────────────────────────────────┐
                    │         메인 (바텀 네비 4탭)          │
                    ├──────┬──────┬──────────┬─────────────┤
                    │ 홈   │ 방   │ 캐릭터   │ 설정         │
                    └──┬───┴──┬───┴────┬─────┴──────┬──────┘
                       │      │        │            │
                       ↓      ↓        ↓            ↓
                    캘린더  방 목록  캐릭터 편집  설정 화면
                    + FAB   + 방 상세
                       │      │
                       ↓      ↓
                   기록 화면  공유 지도
                       │
                       ↓
                  지도 애니메이션
                  (Dottie 핵심 경험)
```

## 네비게이션 구조

```
BottomNavigationBar (4탭)
├── Tab 1: 홈 (타임라인)
│   ├── 캘린더 + 최근 기록 리스트
│   ├── → 날짜 탭 → 지도 애니메이션 화면
│   └── → FAB → 기록 시작 → 기록 화면
│                              └── → dot 찍기 바텀시트
│                              └── → 기록 종료 → 지도 애니메이션
├── Tab 2: 방
│   ├── 방 목록
│   ├── → 방 탭 → 방 상세 (멤버 + 캘린더)
│   │              └── → 날짜 탭 → 합본 지도
│   └── → + 새방 → 방 만들기 → 초대 링크 공유
├── Tab 3: 캐릭터
│   └── 캐릭터 편집기
└── Tab 4: 설정
    ├── 프로필 편집
    ├── 알림 설정
    ├── 위치 권한
    └── 계정 관리
```

## 디자인 가이드라인

### 컬러 팔레트
- **Primary:** #4A90D9 (도티 블루)
- **Secondary:** #FF8A80 (코랄 핑크)
- **Accent:** #B39DDB (라벤더)
- **Background:** #FAFBFC (라이트 그레이)
- **Surface:** #FFFFFF
- **Text Primary:** #1A1A2E
- **Text Secondary:** #6B7280

### 타이포그래피
- **한글:** Pretendard (or 기본 시스템 폰트)
- **영문/숫자:** Pretendard or SF Pro
- **로고:** 커스텀 (dottie 로고타입)

### 디자인 톤
- 둥근 모서리 (border-radius: 16px)
- 부드러운 그림자
- 파스텔 톤 배경
- 미니멀하지만 귀여운 느낌 (setlog 감성 참고)
- 이모지 적극 활용
